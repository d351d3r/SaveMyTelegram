Parser of selected telegram chats and backup on the local storage

## Dependencies
* [TDLib](https://github.com/tdlib/td);
* [Nlohmann JSON](https://github.com/nlohmann/json);
* C++17-compliant compiler 
## Build
```sh
mkdir build; cd build;
cmake ..
```
